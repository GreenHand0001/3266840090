# 项目简介

## 核心功能
- 提供基于YOLOv8的窗户实例分割检测系统完整源码
- 包含2500张标注数据集（仅'Window'一类）
- 提供训练好的模型及训练结果
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供用户界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存功能
- 提供项目运行教程和环境配置指南

## 适用场景
- 建筑物窗户自动化检测
- 智能安防监控系统
- 建筑物立面分析
- 房地产评估
- 城市规划研究
- 建筑物维护管理

## 技术特点
- 基于YOLOv8深度学习框架
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python 3.9、OpenCV、PyQt5、Ultralytics、PyTorch 1.9等技术栈
- 提供图形用户界面，操作简便
- 支持实时检测与结果可视化
